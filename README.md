# LetsEat.flutter


# 🍚  밥 한번 먹자
한국인들이 흔히 하는 인삿말 중 하나인 '언제 밥 한번 먹자'가 있습니다.
흔히 해어질 떄 다시 한번 만나자며 하는 형식적인 약속을 진짜 약속으로 만들고자 하여 진행하게 되었습니다.


## :pushpin: Features
- 로그인/ 자동로그인
- QR코드를 사용한 약속 생성 및 목록 조회
- 콕 찔러보기 기능
- 찔러보기 기록 확인
